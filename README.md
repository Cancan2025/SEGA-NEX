SEGA NEX is fanmade video game console os with 2006 vibe. It’s the "What If?" project for the seventh generation of gaming. The seventh generation saw the release of Microsoft’s Xbox 360, Sony’s PlayStation 3, and Nintendo’s Wii. But what about... SEGA’s game console?  Missing that piece of history? Then it’s time to go to the NEX Level.
